## 가입 계약 검증

- `PROFILE_REQUIRED` 응답은 토큰을 유지한 채 프로필 설정 화면으로 이동한다.
- 신규 가입은 필수 약관 동의와 닉네임 중복 확인 후 완료된다.
- 선택 약관에 동의하지 않아도 가입할 수 있다.
- 프로필 이미지는 선택값이며 기존 파일 검증을 유지한다.
- `AUTHENTICATED` 사용자도 닉네임 또는 필수 약관이 미완료면 가입 화면으로 복구된다.
- 약관 저장 실패 시 프로필을 저장하지 않고 현재 화면에서 재시도할 수 있다.
- 완료 버튼 연타 시 프로필 저장 요청은 한 번만 실행된다.

## 개인정보 검증

- 전화번호 인증 API, 응답 모델, 입력 UI, 타이머, 마스킹 전화번호 표시를 제거했다.
- 성별·생년월일은 가입 및 프로필 수정 화면과 요청에서 제외했다.
- 앱 내 개인정보 처리방침에서 해당 수집·처리 문구를 제거했다.
