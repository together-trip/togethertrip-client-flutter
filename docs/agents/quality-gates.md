# Quality Gates

## 기본 검증 명령

```bash
flutter analyze
flutter test
```

## 보안 체크리스트

- 카카오 로그인 토큰 저장 방식
- 전화번호, 프로필, 생년월일 등 개인정보 표시와 로컬 저장
- 사진 업로드 제한과 미리보기 삭제 처리
- 알림 권한, 딥링크 접근 권한
- 여행/정산 화면의 권한별 버튼 노출

## 완료 기준

- 관련 테스트 또는 분석 명령을 실행했다.
- 실행하지 못한 검증은 이유를 남겼다.
- 사용자 흐름, 권한, 입력 제한이 기획과 충돌하지 않는다.
- 문서 변경이 필요한 경우 함께 갱신했다.
